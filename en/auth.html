SSO Authentication Service

An extensible authentication and identity management service implementing OAuth 2.0 and OpenID Connect standards. Initial deployment involved integration with on-premises IAM providers. The service was later integrated into the internal IaC-based CI/CD platform and connected to the authorization router. Additional integration was performed with the in-house A2P message delivery platform and a security audit/logging service.

Implementation Details

Developed in Golang, the authentication service handles device-bound session management for mobile, web, and desktop clients. It supports registration, authentication, authorization, and full session lifecycle control. The system is optimized for performance and horizontal scalability.

Implemented features include:

  • OAuth 2.0 and OpenID Connect flows
  • JWT issuance (HMAC and RSA)
  • Multi-factor authentication and policy management
  • Username/password login with MFA
  • QR code-based authentication
  • OTP-based authentication
  • Certificate-based authentication (X.509)

The service integrates with: identity providers for account and role management; internal messaging delivery system (for OTP, confirmations); audit and security logging services (event tracking, anomaly detection); GraphQL-based admin interface for managing users, policies, and sessions.

Client-side Authentication

Client authentication logic was implemented for Android (Kotlin with RxJava) and Web/Desktop (React and Electron using RxJS). Supports persistent session management, token refresh workflows, and QR-based login across platforms.
